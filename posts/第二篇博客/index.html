使用 GitHub Pages 预览 HTML

October 19, 2019

GitHub Pages

GitHub Pages 是 GitHub 提供的用于展示托管在 GitHub 上的静态网页的功能，简而言之，通过它我们可以预览当前代码仓库中的所有网页。利用这个功能，用户还可以搭建自己的个人博客。

如何使用GitHub Pages

假设我们已经上传了一个本地仓库到 GitHub 上，我们在 Github 中打开这个仓库，仓库中已经有了一个 HTML 文件，但是我们不能预览，点击进入以后看到的是网页源码。

接下来我们开启 GitHub Pages 功能，点击 Settings，往下翻找到 GitHub Pages，选择 master branch，这时 GitHub Pages 就已经开启了

我们可以看到 GitHub Pages 提供的一个链接

复制这个链接，并在后面加上 index.html (https://leeheart.github.io/pages-test/index.html) 即可预览我们的网页了
